# Kolkie Inkloksysteem

Simpel inklok- en roostersysteem voor Cafetaria Kolkie.

## Tech stack
- **Frontend:** Svelte + Vite
- **Backend:** Rust + Rocket
- **Database:** SQLite

## Opstarten

### Backend
```bash
cd backend
cargo run
```
Draait op http://localhost:8000

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Draait op http://localhost:5173
